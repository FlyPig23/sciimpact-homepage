# SCIIMPACT Homepage

This folder is a clean GitHub Pages package for the SCIIMPACT homepage.

## Included

- `index.html`
- `static/css/index.css`
- `static/js/index.js`
- `static/images/*`
- `.nojekyll`
- `.gitignore`

## Upload To GitHub

1. Create a new GitHub repository.
2. Upload all files in this folder to the repository root.
3. In GitHub, open `Settings` -> `Pages`.
4. Set `Source` to `Deploy from a branch`.
5. Set the branch to `main` and folder to `/ (root)`.
6. Save and wait for GitHub Pages to publish.

## Notes

- The homepage no longer links to a local PDF, since the paper PDF was removed from the package.
- If you later want to add the PDF back, place it in the repository root and add a button link in `index.html`.
- A repository license is not included yet, since it has not been finalized.
